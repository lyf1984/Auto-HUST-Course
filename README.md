# 🎓 Auto-HUST-Course (华科课程平台全自动刷课助手)

华中科技大学（HUST）Smartcourse 课程平台**全自动**视频学习辅助脚本。一键开启，自动连播

[LiuziqiOvO/HUST-Course-Helper](https://github.com/LiuziqiOvO/HUST-Course-Helper) 修改增强。原版脚本实现了单次点击破解当前视频的功能，本增强版在此基础上加入了**后台轮询、弹窗突破、防卡死、全自动翻页**等机制。

## 🚀 安装指南

1. 浏览器安装用户脚本管理器插件，推荐使用 **[Tampermonkey](https://www.tampermonkey.net/)** (油猴)。
2. 点击下方链接一键安装脚本：
   👉 [点击此处安装脚本](https://greasyfork.org/zh-CN/scripts/569003-%E5%8D%8E%E7%A7%91%E8%AF%BE%E7%A8%8B%E5%B9%B3%E5%8F%B0%E8%87%AA%E5%8A%A8%E5%88%B7%E8%AF%BE%E5%8A%A9%E6%89%8B-%E5%85%A8%E8%87%AA%E5%8A%A8%E6%97%A0%E9%98%BB%E6%8B%A6%E7%89%88)

## 📖 使用说明

1. 安装本脚本后，登录华科 Smartcourse 平台。
2. 进入任意一节带有视频的学习页面（网址中包含 `studentstudy`）。
3. 页面右上角会自动出现一个 **【▶ 开启全自动刷课】** 的绿色按钮。
4. 点击开启后，按钮变为红色 **【⏸ 自动刷课中...】**。
5. 去喝杯咖啡 ☕，脚本会自动帮你把剩下的所有章节刷完！当课程全部播放完毕（找不到下一节时），脚本会自动停止。
